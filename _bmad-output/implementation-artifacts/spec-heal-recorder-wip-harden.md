## Design Notes

The sentinel-span pattern is load-bearing: `BaseModal` already uses Vue `Teleport` internally, so a `data-testid` on the modal root does not reliably reach `document.body` for Playwright's `getByTestId`. The sentinel decouples test visibility from the modal's internal DOM. The inline comment above the `<Teleport>` block must survive — it explains a non-obvious WHY.
