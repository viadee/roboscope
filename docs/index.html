In 3 Schritten startklar

Kein Cloud-Account nötig. Alles läuft lokal auf deinem Rechner.

1

Voraussetzungen

Python 3.12+, Node.js 20+ und uv installieren.

curl -LsSf https://astral.sh/uv/install.sh | sh
2

Clone & Install

Repository klonen und Dependencies installieren.

git clone https://github.com/viadee/roboscope.git cd roboscope cp backend/.env.example backend/.env make install
3

Starten

Backend und Frontend mit einem Befehl starten.

make dev # Backend: http://localhost:8000 # Frontend: http://localhost:5173 # Login: admin@roboscope.local / admin123
