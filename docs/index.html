Download ZIP ↓ GitHub → Quick Install

Robot Framework Testmanagement in einer modernen Web-Oberfläche

🚀

Test Execution

Subprocess oder Docker, Live-WebSocket-Updates, Cron-Scheduling — Tests direkt aus dem Browser starten und beobachten.

Projekt & Git Management

Repos klonen, Branches wechseln, automatisch synchronisieren. Integrierter Code-Explorer mit Robot-Parser und CodeMirror-Editor.

Environment Management

Reports & Deep Analysis

Rollen & Zugriff

LLM-Anbieter

In Minuten startklar

Kein Cloud-Account nötig. Alles läuft lokal auf deinem Rechner.

Empfohlen

ZIP Download (Offline)

Lade das ZIP-Archiv herunter, entpacke es und starte mit einem Befehl. Kein Git, kein Node.js, kein Internet nötig.

1

Download & Entpacken

ZIP von GitHub Releases herunterladen und entpacken.

2

Installieren

chmod +x install-mac-and-linux.sh ./install-mac-and-linux.sh
install-windows.bat
3

Starten

./start-mac-and-linux.sh # Öffne http://localhost:8145 # Login: admin@roboscope.local / admin123
start-windows.bat REM Öffne http://localhost:8145 REM Login: admin@roboscope.local / admin123
Entwickler

Git Clone (Dev-Setup)

Für Entwickler, die den Quellcode bearbeiten und zum Projekt beitragen möchten. Benötigt Python 3.12+, Node.js 20+ und uv.

1

Voraussetzungen

# uv installieren (Package Manager) curl -LsSf https://astral.sh/uv/install.sh | sh
2

Clone & Install

git clone https://github.com/viadee/roboscope.git cd roboscope cp backend/.env.example backend/.env make install
3

Dev-Server starten

make dev # Backend: http://localhost:8000 # Frontend: http://localhost:5173 # Login: admin@roboscope.local / admin123
